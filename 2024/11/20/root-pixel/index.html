root Pixel 6a手机

2024/11/20 APP分析

1.1谷歌解锁 Bootloader

1.电脑安装USB驱动：usb_driver_windows

2.手机启动开发者模式(Pixel 6a是在设置-关于手机中疯狂点击版本号，然后便会启动开发者模式)

3.开发者选项 开启 USB调试 与 OEM解锁

4.电脑下载: adb-fastboot

5.手机连接电脑，打开cmd，进入到adb-reboot的目录下，输入指令

adb reboot bootloader

将手机重启到 Bootloader模式。

6.输入解锁指令:

fastboot flashing unlock

7.按手机的 音量下键 滚动选择 Unlock the bootloader，再按电源键 确定 解锁。

8.手机显示 Device state：unlocked 解锁完成。

9.输入指令:

fastboot reboot

重启手机

1.2下载ROM系统包

1.2.1下载地址(记得下载与手机版本号对应的、年份最新的系统包)

1.2.2检查有没有boot.img，如果有，跳到1.3步骤；如果没有，就解压出系统包的payload.bin文件

1.2.3下载 payload-dumper-go

1.2.4复制 payload.bin 文件到 payload-dumper-go 文件夹里面

1.2.5在payload-dumper-go文件夹下打开cmd，根据提示输入b,然后就能在同目录找到img文件夹了。

1.3传输boot.img到手机

1.3.1手机插电脑，文件传输模式，复制 boot.img 到手机 Download 目录

1.3.2如果电脑没反应，也可以用带type-c口的U盘传输文件。

1.4手机下载magisk

1.4.1下载地址

1.5使用magisk安装系统

1.5.1手机打开 magisk app，Magisk图标的右上角的[安装]按钮。

1.5.2选择boot.img，点击[开始]按钮。

1.5.3看到All done!字样，表示系统安装成功。

1.6传输magisk.img到电脑

1.6.1仿造1.3步骤，把手机上在Download生成的magisk.img传输到电脑上adb-fastboot 目录。

1.7将包刷入手机

1.7.1重复1.1谷歌解锁 Bootloader步骤，手机进入 Bootloader 模式，再插入电脑。

1.7.2打开“打开CMD命令行.bat”，输入下面的命令：

fastboot flash boot 你自己的magisk文件名.img

1.73输出下面这三行代码，就是成功刷入了。

Sending 'boot' (131072 KB) OKAY [ 3.105s]
Sending 'boot' (131072 KB) OKAY [ 3.105s]
Writing 'boot'             OKAY [ 0.404s]
Finished. Total time: 3.556s

1.7.4再使用指令:

 fastboot reboot

重启手机。

1.7.5如果出现手机反复重启情况，先同时长按上音量键和关机键进入黑屏，然后迅速在下一次重启前长按下音量键，进入boot loader模式，再输入指令:

fastboot flush boot boot.img

将之前下载的初始boot.img刷入手机，应该就可以重新开机了。
