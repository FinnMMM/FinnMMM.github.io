1.下载 Frida

Windows

在终端输入以下指令来下载frida的基本配置：(在此之前你可能需要先安装python和pip)

pip install frida
pip install frida-tools

然后输入以下指令查看手机的CPU型号：

adb shell getprop ro.product.cpu.abi

然后去这里下载对应CPU型号并且与电脑安装Frida版本号一致的Frida-server（Frida的版本可以用frida --v查看）。
我的Frida版本号是16.5.2，CPU型号是arm64-v8a，选择下载了frida-server-16.5.2-android-arm64.xz。
然后将下载的压缩包解压，重命名为frida-server，并输入以下指令将其上传到手机上：

adb push .\frida-server /data/local/tmp

然后再给其授予777权限。
依次输入以下指令：

adb shell               #进入shell
su root                 #切换到root权限
cd /data/local/tmp      #切换到/data/local/tmp目录
chmod 777 frida-server  #给frida-server文件赋予777权限

至此，Frida的基本配置就完成了。
检验是否安装成功：

adb shell
su root
cd /data/local/tmp
./frida-server

如果不报错也不结束，再新开一个终端，输入以下指令：

frida-ps -U

如果能看到手机上的进程列表，说明安装成功。
